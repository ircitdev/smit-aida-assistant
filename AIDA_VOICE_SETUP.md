## ⚠️ ВАЖНО: Ошибка 3128 - Настройка автоответчика

### Проблема
Команда  возвращает ошибку **3128**, несмотря на правильные параметры.

### Причина
API команда  работает **только для активных разговоров**, когда звонок УЖЕ соединен.
Для входящих звонков в IVR состоянии команда не работает.

### Решение
Необходимо настроить **автоответчик** в личном кабинете Mango Office:

1. Войдите в [app.mango-office.ru](https://app.mango-office.ru)
2. Перейдите: **Настройки** → **Маршрутизация** → **Входящие звонки**
3. Найдите номер **+7(844)255-35-59**
4. Настройте **автоматическое принятие звонка**:
   - Включите опцию Автоответчик
   - ИЛИ настройте маршрутизацию на внутренний номер с автоответом
   - ИЛИ используйте IVR меню с немедленным ответом

### Альтернативные подходы
1. **IVR меню в Mango Office**: Загрузите приветствие напрямую в личном кабинете
2. **SIP интеграция**: Используйте прямое SIP подключение вместо HTTP API
3. **WebRTC**: Подключитесь через WebRTC для полного контроля над звонком
